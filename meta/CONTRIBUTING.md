# Contributing to Words of Istanbul

First of all, thank you for considering contributing to this project! Istanbul is a city of layers, and we need every perspective to document its soul.

## How Can You Contribute?

### 1. Adding New Quotes or Poems
If you have a powerful quote about Istanbul (local or international), please add it to the relevant module in the `core/` directory.
- Ensure the source is verified.
- Provide a brief context if the quote is from a specific historical event or literary work.

### 2. Deepening the Analysis
Help us improve the psychological and historical analysis in modules like `core/huzun-ve-melankoli.md`. Contributions that draw from academic sources or deep literary critiques are highly encouraged.

### 3. Fixing Errors
Spelling mistakes, broken links, or historical inaccuracies—please let us know by opening an issue or a Pull Request.

## Contribution Guidelines

- **Quality over Quantity:** We aim for "High-Density" content. Instead of listing hundreds of generic quotes, we prefer 10 impactful, well-analyzed pieces.
- **Language:** While the primary language is Turkish, we welcome bilingual (TR/EN) contributions to make the city's soul accessible to a global audience.
- **Aesthetics:** If you are adding markdown elements, maintain the professional formatting (emojis, tables, and blockquotes).

## Process

1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/amazing-quote`).
3. Commit your changes (`git commit -m 'Add: Nazim Hikmet - Hasret poetry'`).
4. Push to the branch (`git push origin feature/amazing-quote`).
5. Open a Pull Request.

Thank you for helping us document the "Soul of the City"!
